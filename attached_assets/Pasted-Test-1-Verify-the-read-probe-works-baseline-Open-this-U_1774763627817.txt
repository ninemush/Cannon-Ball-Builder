Test 2: Attempt a write (task catalog creation) via the OData endpoint
Run this curl from your local machine (or Postman). You'll need a fresh token first.

Step 2a — Get a token:

curl -s -X POST "https://cloud.uipath.com/identity_/connect/token" \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "grant_type=client_credentials&client_id=YOUR_CLIENT_ID&client_secret=YOUR_CLIENT_SECRET&scope=OR.Tasks OR.Tasks.Read OR.Tasks.Write"
Copy the access_token from the response.

Step 2b — Try OData create:

curl -s -X POST "https://cloud.uipath.com/uipatezkzunj/DefaultTenant/orchestrator_/odata/TaskCatalogs" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -H "X-UIPATH-OrganizationUnitId: YOUR_FOLDER_ID" \
  -d '{"Name": "CannonBall_AC_WriteTest", "Description": "Temporary test catalog - safe to delete"}'
Record the full response — status code and body.
